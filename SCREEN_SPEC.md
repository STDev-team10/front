# 화면 명세서

## 1. 서비스 개요

화학 원소를 조합해 화합물을 만드는 학습형 게임 서비스이다.  
사용자는 로그인 후 플레이 모드를 선택하고, 난이도별 문제를 풀거나 샌드박스에서 자유롭게 실험할 수 있다.  
문제를 맞히면 도감이 해제되고, 한 판을 완주하면 포인트와 기록이 저장된다.

---

## 2. 화면 구조

### 2-1. 시작 화면

#### 화면명
시작 화면

#### 목적
로그인, 회원가입, 게스트 입장을 선택하는 첫 진입 화면

#### 주요 요소
- 로그인 버튼
- 회원가입 버튼
- 게스트로 시작 버튼

#### 동작
- 로그인 버튼 클릭 시 로그인 화면으로 이동
- 회원가입 버튼 클릭 시 회원가입 화면으로 이동
- 게스트로 시작 클릭 시 모드 선택 화면으로 이동

---

### 2-2. 로그인 화면

#### 화면명
로그인 화면

#### 목적
기존 계정으로 서비스에 로그인

#### 주요 요소
- 아이디 입력창
- 비밀번호 입력창
- 로그인 버튼
- 뒤로가기 버튼
- 에러 메시지 영역

#### 동작
- 로그인 성공 시 모드 선택 화면으로 이동
- 로그인 실패 시 에러 메시지 표시
- 로그인 응답으로 받은 `token`, `username`, `points` 저장

---

### 2-3. 회원가입 화면

#### 화면명
회원가입 화면

#### 목적
신규 계정 생성

#### 주요 요소
- 아이디 입력창
- 비밀번호 입력창
- 회원가입 버튼
- 뒤로가기 버튼
- 에러 메시지 영역

#### 동작
- 회원가입 성공 시 자동 로그인 후 모드 선택 화면으로 이동
- 회원가입 실패 시 에러 메시지 표시
- 회원가입 응답으로 받은 `token`, `username`, `points` 저장

---

### 2-4. 플레이 모드 선택 화면

#### 화면명
플레이 모드 선택 화면

#### 목적
사용자가 원하는 플레이 방식을 선택

#### 주요 요소
- 사용자 닉네임
- 사용자 포인트 표시 버튼
- 로그아웃 버튼
- `normal 모드`
- `hardcore 모드`
- `샌드박스 모드`
- `명예의 전당`

#### 동작
- 닉네임 옆 포인트 버튼 클릭 시 포인트 랭킹 팝업 표시
- `normal 모드` 클릭 시 레벨 선택 화면으로 이동
- `hardcore 모드` 클릭 시 레벨 선택 화면으로 이동
- `샌드박스 모드` 클릭 시 바로 플레이 화면으로 이동
- `명예의 전당` 클릭 시 명예의 전당 화면으로 이동

#### 비고
- 포인트 버튼에는 `⭐ {points}P` 형태로 표시
- 게스트는 포인트 랭킹 버튼 비활성화

---

### 2-5. 레벨 선택 화면

#### 화면명
레벨 선택 화면

#### 목적
선택한 모드에서 난이도를 고르는 화면

#### 주요 요소
- 사용자 닉네임
- 사용자 포인트 표시 버튼
- 모드 선택으로 돌아가기 버튼
- 로그아웃 버튼
- 난이도 카드 4개
  - LEVEL 1: `초보 (순한맛)`
  - LEVEL 2: `중수 (보통맛)`
  - LEVEL 3: `고수 (매운맛)`
  - LEVEL 4: `전문가 (기억력 테스트)`
- 각 난이도별 왕관 버튼
- 화합물 도감 버튼
- 안내 문구

#### 동작
- 난이도 카드 클릭 시 게임 시작
- 왕관 버튼 클릭 시 해당 난이도의 타임어택 랭킹 팝업 표시
- 포인트 버튼 클릭 시 포인트 랭킹 팝업 표시
- 화합물 도감 버튼 클릭 시 도감 화면으로 이동

#### 비고
- 각 난이도 카드에는 개인 최고 기록 표시
- 문제 수 고정
  - easy: 5개
  - medium: 10개
  - hard: 15개
  - mimic: 20개

---

### 2-6. 게임 플레이 화면

#### 화면명
게임 플레이 화면

#### 목적
원소를 조합하여 화합물을 만드는 메인 플레이 화면

#### 주요 요소
- 상단 정보 영역
  - 일반/하드코어: 하트, 난이도 표시, 점수
  - 샌드박스: 되돌아가기 버튼, 도감 버튼, SANDBOX 표시
- 목표 화합물 정보
  - 화합물 이름
  - 화학식
- 원소 목록 영역
- 트레이 영역
- 초기화 버튼
- `맛있게 비비기!` 버튼

#### 동작
- 원소 클릭 또는 드래그 시 트레이에 추가
- 트레이 원소 제거 가능
- `맛있게 비비기!` 클릭 시 정답 판정
- 일반 모드: 이름 + 화학식 표시
- 하드코어 모드: 이름만 표시, 화학식 숨김
- 샌드박스 모드: 문제 없이 자유 조합

#### 비고
- 하드코어 모드 하트 수: 1개
- 일반 모드 하트 수: 3개
- 하드코어는 스테이지당 점수 보상이 더 큼

---

### 2-7. 미믹 프리뷰 화면

#### 화면명
미믹 프리뷰 화면

#### 목적
기억력 테스트 난이도에서 잠깐 목표를 보여준 뒤 숨김 처리

#### 주요 요소
- 목표 화합물 정보
- 카운트다운

#### 동작
- 일정 시간 카운트다운 후 실제 플레이 화면으로 전환

---

### 2-8. 정답/실패/게임 종료 모달

#### 화면명
결과 모달

#### 목적
문제 결과와 다음 행동 제공

#### 유형
- 정답 모달
- 실패 모달
- 게임 오버 모달
- 게임 완료 모달

#### 정답 모달 주요 요소
- 분자 뷰어 또는 이모지
- 화합물 이름
- 화학식
- 설명 문구
- AI 설명
- 다음 버튼

#### 정답 모달 동작
- 일반 플레이 중 정답이면 다음 스테이지 또는 메뉴 이동
- 샌드박스에서 발견 시 `계속 실험하기` 버튼 표시
- AI 설명은 `AI선생님:` 라벨과 함께 표시

#### 실패 모달 주요 요소
- 실패 아이콘
- 정답 원소 조합 안내
- 다음 문제 버튼

#### 게임 오버 모달 주요 요소
- 게임 오버 아이콘
- 정답 원소 조합 안내
- 메뉴로 돌아가기 버튼

#### 게임 완료 모달 주요 요소
- 완료 아이콘
- 완료 메시지
- 메뉴로 돌아가기 버튼

#### 비고
- 피가 모두 소진되면 `게임 오버`
- 마지막 스테이지 클리어 시 `게임 완료`

---

### 2-9. 타임어택 랭킹 팝업

#### 화면명
타임어택 랭킹 팝업

#### 목적
난이도별 상위 기록과 내 순위를 조회

#### 주요 요소
- 난이도명
- 상위 랭킹 목록
- 내 현재 순위
- 닫기 버튼

#### 동작
- 레벨 선택 화면에서 왕관 버튼 클릭 시 표시
- 상위 5명 또는 10명 랭킹 표시
- 내 기록이 있으면 내 순위와 기록 시간 표시

---

### 2-10. 포인트 랭킹 팝업

#### 화면명
포인트 랭킹 팝업

#### 목적
전체 포인트 상위 랭킹과 내 순위를 조회

#### 주요 요소
- 상위 5명 목록
- 내 현재 순위
- 닫기 버튼

#### 동작
- 닉네임 옆 포인트 버튼 클릭 시 표시
- 상위 5명 포인트 표시
- 내 포인트와 현재 순위 표시

---

### 2-11. 화합물 도감 화면

#### 화면명
화합물 도감 화면

#### 목적
사용자가 해제한 화합물 목록 확인

#### 주요 요소
- 뒤로가기 버튼
- 난이도별 섹션
- 화합물 카드 목록
- `+N` 카드

#### 동작
- 난이도별 상위 5개 카드 우선 표시
- 6번째 위치에 남은 개수 `+N` 표시
- `+N` 클릭 시 나머지 카드 펼침
- 다시 클릭 시 접기

#### 비고
- 일반/하드코어 모드에서는 해당 문제의 정답 화합물만 해제
- 샌드박스에서는 실제 조합 성공 시 해당 화합물 해제

---

### 2-12. 명예의 전당 화면

#### 화면명
명예의 전당 화면

#### 목적
특별 발견 이미지와 발견 유저 목록을 보여주는 화면

#### 주요 요소
- 뒤로가기 버튼
- 이미지 카드 목록
- 발견 여부 표시
- 팝업

#### 동작
- 누군가 발견한 항목은 실제 이미지 노출
- 아무도 발견하지 못한 항목은 `?` 와 `미발견` 표시
- 발견된 항목 클릭 시 발견 유저 이름 리스트 팝업 표시

#### 이미지 순서
1. 귀금속
2. 다이너마이트
3. 라듐
4. 락스
5. 비소
6. 살충제
7. 석면
8. 알콜
9. 수은
10. 하버

## 3. 게임 규칙

### 3-1. 모드별 규칙

#### normal 모드
- 일반적인 문제 풀이 방식
- 하트 3개
- 화학식 표시

#### hardcore 모드
- 하트 1개
- 화학식 숨김
- 화합물 이름만 표시
- 더 높은 점수 획득

#### 샌드박스 모드
- 목표 문제 없음
- 자유 조합 가능
- 조합 성공 시 도감 해제 가능

---

## 4. 점수/포인트 정책

### 4-1. 점수
- 일반 모드: 스테이지당 기본 점수 획득
- 하드코어 모드: 일반보다 높은 점수 획득

### 4-2. 포인트
- 한 문제 정답마다 저장하지 않음
- 한 판 전체를 끝까지 완주했을 때 최종 점수만큼 포인트 적립
- 로그인 유저는 서버에 포인트 저장
- 게스트는 로컬 세션 기준 포인트 저장

---

## 5. 기록 정책

### 5-1. 타임어택 기록
- 게임 시작 시 시각 저장
- 마지막 스테이지를 클리어한 순간 총 소요 시간 계산
- 해당 난이도 1판 전체 완주 시간을 기록으로 저장

### 5-2. 기록 대상
- normal / hardcore 만 대상
- sandbox 제외
- 게스트 제외

---

## 6. API 연동 요약

### 인증
- 로그인: `/api/auth/login`
- 회원가입: `/api/auth/signup`
- 내 정보 조회: `/api/auth/me`

### 포인트
- 포인트 저장: `POST /api/auth/points`
- 포인트 랭킹: `GET /api/auth/points-rankings`

### 화합물
- 화합물 목록: `GET /api/compounds`
- 도감 해제 목록: `GET /api/compounds/unlocks/me`
- 화합물 해제 저장: `POST /api/compounds/{compound_id}/unlock`

### 타임어택
- 기록 저장: `POST /api/compounds/time-attack-records`
- 랭킹 조회: `GET /api/compounds/time-attack-rankings`

---

## 7. 비고

- 프론트는 상대 경로 `/api/...` 방식 사용
- 개발 환경과 배포 환경 모두 프록시/리라이트 기반으로 API 연결
- 포인트, 도감, 랭킹은 로그인 유저 기준 서버 연동 가능
- 게스트는 일부 기능이 제한됨
